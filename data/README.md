# Data Directory
This directory contains the raw and processed data for the long document summarization project.

## Usage

### 1. Process Raw PDFs

Run the batch processor to extract and clean text from all PDFs:

```bash
cd data
python process_pdfs.py
```

This will:
- Extract text from all PDFs in `raw/`
- Clean and normalize the text
- Extract metadata (title, abstract, arXiv ID, etc.)
- Save cleaned text to `processed/*.txt`
- Save metadata to `processed/*_metadata.json`
- Generate a processing summary in `processed/processing_summary.json`

### 2. Access Processed Documents

Use the `DocumentDataset` class to work with processed documents:

```python
from data.dataset import DocumentDataset

# Load dataset
dataset = DocumentDataset()

# Get statistics
stats = dataset.get_statistics()
print(f"Total documents: {stats['total_documents']}")

# List all documents
for doc_info in dataset.list_documents():
    print(doc_info['title'])

# Get a specific document
doc = dataset[0]  # By index
text = doc['text']
metadata = doc['metadata']

# Get by ID
doc = dataset.get_by_id('2312.16337v1')

# Get by arXiv ID
doc = dataset.get_by_arxiv_id('2312.16337')

# Get random document for testing
doc = dataset.get_random_document()

# Filter by length
long_docs = dataset.filter_by_length(min_words=5000)

# Search in titles
results = dataset.search_in_titles('summarization')
```

### 3. Run from Jupyter Notebook

```python
# In notebooks/
import sys

doc = dataset[0]
print(doc['metadata']['title'])

```

## Data Files

### Raw Directory (`raw/`)
Contains original PDF files of research papers. These are typically arXiv papers identified by their IDs (e.g., `2312.16337v1.pdf`).

### Processed Directory (`processed/`)
After running `process_pdfs.py`, contains:

- **Text files** (`*.txt`): Clean, extracted text from PDFs
- **Metadata files** (`*_metadata.json`): Document metadata including:
  - filename
  - arxiv_id
  - title (extracted)
  - abstract (extracted)
  - processed_date
  - word_count
  - char_count
- **processing_summary.json**: Summary of the batch processing run
- **dataset_summary.json**: Overall dataset statistics (generated by `dataset.py`)

## Metadata Structure

Example metadata JSON:

```json
{
  "filename": "2312.16337v1.pdf",
  "arxiv_id": "2312.16337",
  "title": "Efficient Long Document Summarization with Transformers",
  "abstract": "We propose a novel approach...",
  "processed_date": "2026-01-08T10:30:00",
  "word_count": 8450,
  "char_count": 52300
}
```

## Notes

- The processor attempts to extract paper titles and abstracts automatically
- Text cleaning removes excessive whitespace and normalizes formatting
- ArXiv IDs are extracted from filenames when available
- All text is saved in UTF-8 encoding
